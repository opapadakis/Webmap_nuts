Light webmap framework using csv to store data pipelines.

Data: Eurostat API at regional level (nuts 2)
Indicator list: https://ec.europa.eu/eurostat/web/main/data/database
API link https://ec.europa.eu/eurostat/web/user-guides/data-browser/api-data-access/api-getting-started

Reproduction:
1. Clone the project
2. Go to Maptiler https://cloud.maptiler.com/account/keys/ and create a key
3. Copy the key in config.js

Use: Click on the buttons to download the data first
